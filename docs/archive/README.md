# Archive Index

Archives preserve completed iteration context, deferred seeds, and old rationale. They are not default session context.

Read an archive only when reactivating an old focus, checking historical rationale, or following a link from a task-specific doc.

## Naming Convention

```text
YYYY-MM-short-topic-kind.md
```

Kinds:

- `seed` — archived idea or roadmap that may be reactivated.
- `iteration` — completed implementation phase.
- `decision` — focused rationale for a route, schema, or contract choice.
- `postmortem` — lessons from a failed or reverted direction.
